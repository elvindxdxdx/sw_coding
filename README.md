# 나만의 과제 일정 관리 프로그램 DoItNow
## 이 프로그램은 고려대학교 학생을 위해 과제 수행의 구체적인 단계들을 제시해주는 과제 일정 관리 프로그램입니다.
### 환영합니다!

**고려대학교 학생**을 위한 멋진 과제 세부 이행 일정 생성법

-먼저, 왼쪽 상단의 톱니바퀴를 눌러 gemini API Key를 설정합니다.

-그 다음, 그 아래에 있는 '새 과제 등록'을 누릅니다.

-입력창에 과제명 입력, 마감일 설정(숫자를 직접 클릭해서 변경할 수 있습니다), 과제 유형 선택(레포트/에세이, 프로그래밍/팀 프로젝트, 시험 공부/자격증, 발표자료/ 기타과제 중 택 1), AI 일정 스타일 선택, 과제 상세 설명 및 요구 사항 작성을 완료합니다.

> AI 일정 스타일 선택은 이용자의 성향에 따라 균등 분할, 초반 집중형, 벼락치기 특화형 중 선택할 수 있습니다.

-세부 이행 일정이 생성됩니다.(하나의 과제를 작은 과제들로 나눈 것처럼 수행해야 하는 각 단계에 대해 D-Day가 부여됩니다.)

-세부 이행 일정의 단계를 완료했다면 그 단계를 클릭하면 옆의 과제 진행률이 상승합니다.
